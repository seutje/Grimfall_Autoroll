# Grimfall Autoroll

Grimfall Autoroll is a World of Warcraft addon that rolls Destiny's Dice until your selected abilities are available.

## Installation

1. Download the project as a ZIP from GitHub.
2. Extract the ZIP into your WoW `Interface/AddOns` folder.
3. Rename the extracted folder to `Grimfall_Autoroll`.
4. Start or reload the game.

## Usage

1. Type `/gfar` in chat to open the addon.
2. Select up to 4 abilities you want to roll for.
3. Play Peggle to roll until the desired abilities are available.
